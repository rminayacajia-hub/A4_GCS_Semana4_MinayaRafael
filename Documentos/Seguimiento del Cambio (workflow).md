## 5. Seguimiento del Cambio (Workflow)

| Estado | Quién | Condición de salida (evidencia) |
| :--- | :--- | :--- |
| **Registrado** | Solicitante/PM | CR creado y descripción completa. |
| **En análisis** | Líder técnico/QA | Impacto y riesgos documentados. |
| **Aprobado** | CCB/PM | Acta de decisión firmada en el sistema. |
| **En implementación** | Dev | Branch/PR con commits vinculados. |
| **Verificado** | QA | Reporte de pruebas aprobado y checklist. |
| **Cerrado** | PM/QA | Release tag v1.4.0 y ticket cerrado. |
